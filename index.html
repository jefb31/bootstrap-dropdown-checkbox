Installation

1. Add jQuery and Bootstrap.

2. Add this after HTML body and jQuery:

<script type"text/javascript" src="/path/bootstrap-dropdown-checkbox.js"></script>

3. Add this after the <head> section:

<link rel="stylesheet" type="text/css" href="/path/bootstrap-dropdown-checkbox.css">

You're done! Good job :)

Usage

The HTML template you should use:

<div class="myDropdownCheckbox"></div>

Then, through a jQuery selector:

var myData = [{id: 1, label: "Test" }];
$(".myDropdownCheckbox").dropdownCheckbox({
  data: myData,
  title: "Dropdown Checkbox"
});

Options

Parameter Type Default Required Description
data Array [] Yes

Mandatory fields:

  • id
  • label
  • isChecked (optional)
autosearch Boolean false No Allows to search the list after each keypress.
sort Function Nothing No Allows a custom sorting function.
sortOptions Anything Undefined No Options when sorting.
title string Undefined Yes (unless you use your own template for button) Title for button.
btnClass string Undefined No CSS class to customize the button.
hideHeader boolean Undefined No Hide header (searchbox).
templateButton string Undefined No Customise your button. You need to have these attributes on your element:
- class: "dropdown-checkbox-toggle"
- data-toggle: "dropdown-checkbox"
showNbSelected boolean false No Display the number of selected items on the button.
An element with the class "dropdown-checkbox-nbselected" must exists.

Methods

Items

Returns the entire list of items.

$("myselector").dropdownCheckbox("item");

Checked

Returns the list of items which are checked.

$("myselector").dropdownCheckbox("checked");

Unchecked

Returns the list of items which are unchecked.

$("myselector").dropdownCheckbox("unchecked");

Append

Appends an array of items.

$("myselector").dropdownCheckbox("append", [{ ... }]);

Appends one item.

$("myselector").dropdownCheckbox("append", { ... });

Remove

Removes multiple items from an array of ids

$("myselector").dropdownCheckbox("remove", [34, 56, 76]);

Reset

Resets the dropdown from an array of items.

$("myselector").dropdownCheckbox("reset", [{ ... }}]);

FAQ

What is an "item" ?

An item contains three fields: id (integer), label (string), isChecked (boolean).
